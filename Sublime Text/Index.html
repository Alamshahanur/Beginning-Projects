This is css (@media) practice

"@media" is one of the css property which is used for a responship wesite and today's maximum of amount of wesite is used this css '@media' property for the good and responship webpage.

HOW TO RIGHT THIS PROPERTY

  1. @media (width:600px){ div{ background-color:red;} }
  2. @media (min-width : 600px) { div { background-color : red; } }
  3. @media (min-width : 200px) and (min-width : 300px) { div { background-color : red; } }
